# 🚀 Namaste React Journey – Complete Learning & Project Repository

This repository contains my complete learning journey through the **Namaste React course**, along with hands-on implementation of all concepts covered across episodes.

The goal of this project is to build a **production-ready React application** while understanding core concepts, advanced patterns, and real-world development practices.

---

# 📌 Tech Stack

* React.js
* JavaScript (ES6+)
* Parcel (Bundler)
* Tailwind CSS
* Redux Toolkit
* React Router
* Jest + React Testing Library

---

# 📚 What This Repository Covers

This project is built step-by-step following all episodes and includes:

* React fundamentals
* Component-based architecture
* Hooks & lifecycle
* Routing
* API integration
* Performance optimization
* State management (Context + Redux)
* Testing (Unit & Integration)
